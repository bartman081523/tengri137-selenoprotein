### 4. C-Translation (Sec→Cys) (Stufe 19)

**Die Übersetzungsregel (implizit in der Alphabet-Beschränkung):**
```
U → C  (Selenocystein → Cystein, Schwefel statt Selen)
O → K  (Pyrrolysin → Lysin)
B → N  (Asparagin statt unspezifiziert)
Z → E  (Glutamat statt unspezifiziert)
J → L  (Leucin statt unspezifiziert)
```

**Begründung:** C fehlt in BURUMUT (Stufe 13, 14) → zeigt, dass das "fremde" Alphabet Sec statt Cys hat. Schmehs "CORRECT GENETIC CODING" = Anwendung der biochemischen Übersetzung.

**C-Übersetzung der BURUMUT-Sequenz (154 AS):**
